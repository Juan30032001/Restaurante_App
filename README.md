## 📝 Descripción
Sistema básico desarrollado en Python aplicando Programación Orientada a Objetos. Permite registrar, almacenar y consultar información de productos del menú y clientes del restaurante, organizado de forma modular para facilitar su uso y mantenimiento.

---

## 🧩 Tipos de Datos Utilizados
- *str*: Para datos de texto como nombres, categorías, correos electrónicos y números de teléfono.
- *float*: Para valores numéricos con decimales, como el precio de los productos.
- *bool*: Para estados lógicos: disponibilidad de platos y condición de miembro del cliente.
- *list*: Estructura de datos para agrupar y almacenar conjuntos de productos y clientes.

---

## ✅ Buenas Prácticas Aplicadas
- Uso de identificadores claros y descriptivos.
- Convención *PascalCase* para nombres de clases.
- Convención *snake_case* para archivos, variables y métodos.
- Código organizado por funciones y responsabilidades.
- Uso adecuado de tipos de datos y estructuras para agrupar información.
